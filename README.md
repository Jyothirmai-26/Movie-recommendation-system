# Movie-recommendation-system
🎬 Movie Recommendation System
📌 Overview
A machine learning project that recommends movies based on similarity using a content-based filtering approach. It analyzes features like genres, cast, keywords, and overview to suggest relevant movies.
🚀 Features
Recommend similar movies 🎯
Uses NLP techniques 🧠
Fast and simple model ⚡
Can be deployed with Streamlit 💻
🛠️ Tech Stack
Python, Pandas, NumPy, Scikit-learn, NLTK, Streamlit
📂 Dataset
tmdb_5000_movies.csv
tmdb_5000_credits.csv
Includes movie details like title, genres, cast, keywords, and overview.
⚙️ Working
1. Data preprocessing (clean & combine features)
2. Create tags (important text data)
3. Convert text → vectors (CountVectorizer)
4. Compute similarity (Cosine Similarity)
5. Recommend top 5 similar movies
▶️ Usage
recommend("Avatar")
